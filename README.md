# 🏛️ Stefany Prodossimo - Arquitetura e Interiores

Landing page desenvolvida para apresentar o portfólio, os serviços e captar leads para o estúdio de arquitetura. O site foi construído com foco em alta performance, design responsivo e conversão direta para o WhatsApp.

## 🚀 Tecnologias Utilizadas

Este projeto é uma **Single Page Application (Página Estática)** leve e otimizada, construída sem a necessidade de frameworks pesados ou steps de build complexos:

* **HTML5:** Estrutura semântica.
* **Tailwind CSS (via CDN):** Estilização rápida, responsiva e moderna.
* **JavaScript Vanilla:** Interações de interface (carrossel de projetos, menu mobile, animações de scroll e montagem do link dinâmico do WhatsApp).
* **Phosphor Icons:** Biblioteca de ícones.
